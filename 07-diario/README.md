# 07 — Diário

Notas diárias escritas pelo assistente após as conversas (destilação), organizadas por ano: `2026/2026-07-03.md`. Usar o template `00-meta/templates/nota-diaria.md`.

Também vivem aqui as reflexões semanais: `2026/reflexao-YYYY-MM-DD.md` (geradas pelo job de sábado, template `reflexao-semanal.md`).

Este é o histórico bruto que alimenta a memória do assistente — é aqui que ele busca evidências para celebrar (ou contra-argumentar) com fatos.
