# 08 — Importado (área de triagem)

Área de chegada de conteúdo externo, antes da triagem para as pastas definitivas:

- `notion-export/` — export do Notion (Settings → Export content → Markdown & CSV). Depois de copiar aqui, rodar a triagem com o assistente: ele categoriza cada nota e propõe mover para a área correta.
- `vault-trabalho/` — cópia do vault do PC do trabalho. Mesma dinâmica de triagem.

Nada aqui é considerado "verdade atual" pelo assistente até ser triado — ele trata este diretório como material de referência desatualizado.
