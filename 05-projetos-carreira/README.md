# 05 — Projetos e carreira

O que vive aqui:
- `metricas.md` — projetos ativos e skills em desenvolvimento (frontmatter).
- `projetos/` — uma pasta ou arquivo por projeto pessoal/profissional, com objetivo, status e próximos passos.
- `skills.md` (criar no onboarding) — habilidades técnicas atuais e as que quer desenvolver.

O assistente rastreia: progresso de cada projeto ativo (cobra projetos parados), evolução de skills e oportunidades de carreira mencionadas nas conversas.
