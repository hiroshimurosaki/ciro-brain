# 02 — Saúde e treino

O que vive aqui:
- `metricas.md` — frontmatter com os números atuais (peso, treinos/semana, sono). O assistente atualiza sempre que o Fernando reportar algo por Telegram.
- `log-treinos/` — um arquivo por treino, nomeado `YYYY-MM-DD.md` (tipo de treino, exercícios, como se sentiu).
- `progresso.md` (criar quando houver dados) — evolução ao longo do tempo.

O assistente rastreia: consistência de treino vs. meta, sono e energia (reportados nas notas diárias), e cobra quando a frequência cair.
