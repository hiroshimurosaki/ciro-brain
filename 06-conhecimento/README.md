# 06 — Conhecimento

Conceitos, insights e aprendizados extraídos das conversas. Criados pelo assistente (via destilação) usando o template `00-meta/templates/aprendizado.md`.

Convenções:
- `conceitos/` — um arquivo por conceito, nome em kebab-case (`regressao-linear.md`).
- Criar subpastas por tema (`livros/`, `artigos/`, `insights/`) quando fizer sentido.
- Sempre linkar com `relacionado:` no frontmatter e `[[links]]` no corpo — conhecimento isolado é conhecimento perdido.
- O briefing matinal usa esta pasta para sugerir "1 coisa para aprender hoje".
